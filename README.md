# 💰 Assistente Financeiro Inteligente

Sistema inteligente desenvolvido com **Python + Streamlit** para análise de gastos, geração de insights financeiros e interação com assistente baseado em IA.

---

## 🚀 Visão geral

Este projeto simula um **assistente financeiro inteligente**, capaz de ajudar o usuário a entender melhor seus hábitos de consumo e tomar decisões financeiras mais conscientes com base em dados reais ou simulados.

A aplicação transforma dados em informações visuais e recomendações práticas.

---

## 🧠 Funcionalidades

- 📊 Análise de gastos por categoria
- 💡 Geração automática de insights financeiros
- 🤖 Chat com assistente inteligente
- 📈 Visualização de dados com gráficos interativos
- 🎨 Interface moderna com CSS personalizado

---

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- JSON
- HTML/CSS
- (Opcional) OpenAI API
