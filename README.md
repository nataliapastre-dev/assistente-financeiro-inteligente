# 💰 Assistente Financeiro Inteligente

Sistema inteligente desenvolvido com **Python + Streamlit** para análise de gastos, geração de insights financeiros e interação com um assistente virtual baseado em IA.

---

## 🚀 Visão geral do projeto

O **Assistente Financeiro Inteligente** é uma aplicação web que simula um sistema de consultoria financeira pessoal.

Ele foi desenvolvido para ajudar o usuário a:

- Entender seus padrões de consumo
- Visualizar onde o dinheiro está sendo gasto
- Receber insights automáticos sobre sua vida financeira
- Interagir com um assistente inteligente em linguagem natural

A proposta do projeto é transformar dados financeiros simples em **decisões mais conscientes e estratégicas**, utilizando análise de dados e automação com Python.

---

## 🧠 Como o sistema funciona

O sistema utiliza uma base de dados simulada contendo:

- Histórico de transações financeiras (CSV)
- Perfil do investidor (JSON)

Com esses dados, a aplicação:

1. Lê e processa as informações financeiras
2. Agrupa gastos por categoria
3. Calcula totais e padrões de consumo
4. Gera insights automáticos
5. Permite interação via chat com o usuário
